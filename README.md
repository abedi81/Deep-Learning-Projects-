# Deep Learning Projects

## Overview

This repository focuses on the fundamental concepts of deep learning through implementing neural networks from scratch.

The current project demonstrates how neural networks learn using forward propagation, backpropagation, and gradient descent, without relying on high-level libraries.

---

## Project

### Neural Network From Scratch

A NumPy-based implementation of a fully connected neural network trained on nonlinear tasks (like XOR) to illustrate how deep learning models capture complex patterns.
